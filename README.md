## About

Fried Frontiers is your go-to resource for discovering the best crispy, golden, and delicious fried food establishments in the beautiful Yarra Valley and Yarra Ranges region. From traditional fish & chips to modern cafeterias and specialty fry spots, we celebrate the art of perfectly fried cuisine.

## Local Development

```bash
npm install
npm run dev
```

The development server runs on port 3001 (Flowise occupies port 3000).
